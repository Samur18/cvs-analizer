=====================================================
  CÓMO ACTUALIZAR EL PROGRAMA
=====================================================

Si hay una nueva versión del Analizador de Evaluaciones,
sigue estos pasos para actualizar sin perder tus datos.


ANTES DE ACTUALIZAR
-------------------
1. Verifica tu versión actual:
   - Abre el archivo VERSION.txt
   - Compara con la versión nueva disponible

2. Lee los cambios:
   - Abre CHANGELOG.md
   - Revisa qué cambió en la nueva versión


MÉTODO 1: DESCARGA COMPLETA (Recomendado)
------------------------------------------
Este es el método más seguro y simple.

PASO 1: Respaldar tus datos
   - Copia todos tus archivos .csv a otra carpeta
   - Ejemplo: Copia a "Mis Documentos\Backup CSV"

PASO 2: Descargar nueva versión
   - Descarga el archivo ZIP de la nueva versión
   - Guárdalo en tu carpeta de Descargas

PASO 3: Instalar nueva versión
   - Opción A: Elimina la carpeta antigua del programa
   - Opción B: Renombra la carpeta antigua a "backup-v1.0"
   - Descomprime el nuevo ZIP

PASO 4: Restaurar tus datos
   - Copia tus archivos .csv de vuelta a la nueva carpeta
   - Ponlos en cualquier ubicación (o crea carpeta "datos")

PASO 5: Reinstalar dependencias
   - Windows: Doble clic en instalar.bat
   - Linux/Mac: Ejecuta ./instalar.sh
   - (Esto actualiza las librerías si hay cambios)

PASO 6: Ejecutar
   - Windows: Doble clic en ejecutar.bat
   - Linux/Mac: Ejecuta ./ejecutar.sh


MÉTODO 2: ACTUALIZACIÓN PARCIAL
--------------------------------
Solo si sabes qué archivos cambiaron.

PASO 1: Ver qué cambió
   - Abre CHANGELOG.md de la nueva versión
   - Busca la sección "Archivos modificados"

PASO 2: Descargar archivos específicos
   - Descarga solo los archivos que cambiaron
   - Ejemplo común: analizador_evaluaciones.py, requirements.txt

PASO 3: Reemplazar archivos
   - Copia los nuevos archivos a la carpeta del programa
   - Confirma que quieres reemplazar los antiguos

PASO 4: Actualizar dependencias
   - Windows: Ejecuta instalar.bat
   - Linux/Mac: Ejecuta ./instalar.sh

PASO 5: Ejecutar
   - Windows: Ejecuta ejecutar.bat
   - Linux/Mac: Ejecuta ./ejecutar.sh


¿QUÉ ARCHIVOS NUNCA BORRAR?
----------------------------
Estos archivos contienen TUS datos o configuraciones:

✅ NUNCA BORRES:
   - Tus archivos .csv (TUS DATOS)
   - Carpeta venv/ (si existe, tiene las librerías instaladas)
   - Cualquier archivo que hayas creado tú

⚠️ PUEDES REEMPLAZAR SIN PROBLEMA:
   - analizador_evaluaciones.py (el programa)
   - README.md, GUIA_COMPLETA.md (documentación)
   - instalar.bat, ejecutar.bat (scripts)
   - instalar.sh, ejecutar.sh (scripts)
   - requirements.txt (lista de librerías)
   - CHANGELOG.md, VERSION.txt (info de versión)

VERIFICAR QUE LA ACTUALIZACIÓN FUNCIONÓ
----------------------------------------
Después de actualizar:

1. Ejecuta el programa (ejecutar.bat o ejecutar.sh)
2. Carga un archivo CSV de prueba
3. Genera los gráficos
4. Exporta a Excel

Si todo funciona correctamente, la actualización fue exitosa.


SOLUCIÓN DE PROBLEMAS
----------------------

Problema: "Error al cargar archivo después de actualizar"
Solución:
   1. Ejecuta instalar.bat nuevamente
   2. Verifica que tus archivos .csv no estén corruptos
   3. Intenta con otro archivo .csv

Problema: "El programa no abre después de actualizar"
Solución:
   1. Verifica que Python siga instalado (python --version)
   2. Ejecuta instalar.bat para reinstalar librerías
   3. Si falla, descarga la versión completa de nuevo

Problema: "Dice que falta una librería"
Solución:
   1. Abre terminal/cmd
   2. Ejecuta: pip install -r requirements.txt
   3. Vuelve a intentar ejecutar el programa

Problema: "Perdí mis archivos CSV"
Solución:
   1. Verifica la carpeta de respaldo que creaste
   2. Busca en "Mis Documentos\Backup CSV"
   3. Si no hiciste respaldo, mira en la papelera de reciclaje
